## リポジトリ追加対象にする

- addするとindexに行く
- ドットを指定したときはカレントディレクトリ上のファイルを全て対象
```console
git add .
```
- 追加対象になったことの確認
```console
git status
```

<pre>
On branch master

Initial commit

Changes to be committed:
  (use "git rm --cached <file>..." to unstage)

          new file:   sample.rb
          </pre>
